# 牙間刷尺寸怎麼選？新手必看完整攻略｜TePe 色碼對照與選購建議

牙間刷是維護口腔健康的關鍵工具，但許多新手最常遇到的問題就是——尺寸到底怎麼選？選錯尺寸不僅無法有效清潔，還可能傷害牙齦。本篇將帶你從原理到實戰，一次搞懂牙間刷選購的所有眉角。

## 為什麼選對尺寸這麼重要？

牙間刷的清潔原理，是靠刷毛與牙面之間的摩擦來移除牙菌斑。如果尺寸太小，刷毛無法充分接觸牙面，清潔力大打折扣；如果尺寸太大，硬塞進牙縫會摩擦損傷牙齦，甚至造成牙根磨損。

## 選購口訣：鐵絲小於縫、刷毛有摩擦

記住這個核心原則：**鐵絲直徑必須小於齒縫寬度，而刷毛在進入牙縫時應該感到輕微的摩擦感**。如果完全沒有阻力，代表尺寸太小；如果需要用力才能推入，代表尺寸太大，應立即換小一號。
